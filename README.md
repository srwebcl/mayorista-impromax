# Mayorista Impromax - Plataforma de Automatización Industrial

Este proyecto es una aplicación web moderna desarrollada con **Next.js 15+ (App Router)** y **Tailwind CSS v4**, diseñada para servir como plataforma de distribución mayorista de componentes de automatización industrial.

## 🛠 Stack Tecnológico

- **Framework**: [Next.js](https://nextjs.org/) (React, TypeScript)
- **Estilos**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Fuentes**: `next/font/google` (Roboto)
- **Iconos**: FontAwesome (CDN)
- **Linting**: ESLint

## 🚀 Instalación y Desarrollo

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/srwebcl/mayorista-impromax.git
    cd mayorista-impromax
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Ejecutar servidor de desarrollo:**
    ```bash
    npm run dev
    ```
    La aplicación estará disponible en `http://localhost:3000`.

4.  **Compilar para producción:**
    ```bash
    npm run build
    npm start
    ```

## 📂 Estructura del Proyecto

- `/app`: Rutas, Layouts y Componentes de página (App Router).
- `/app/components`: Componentes UI reutilizables (Header, Footer, Cards, Forms).
- `/app/globals.css`: Configuración global de estilos y variables CSS.
- `/public`: Activos estáticos (imágenes, iconos).
